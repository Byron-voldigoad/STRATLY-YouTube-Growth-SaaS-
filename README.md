# 🚀 Stratly — YouTube Growth SaaS

**Stratly** est une plateforme SaaS d'analyse et de croissance pour les créateurs YouTube, propulsée par l'intelligence artificielle.

> ⚠️ **Ce projet est un logiciel propriétaire.** Toute reproduction, distribution ou utilisation non autorisée est strictement interdite. Voir la section [Licence](#-licence) ci-dessous.

---

## 🎯 Fonctionnalités

- **📊 Analytics détaillés** — Suivi des abonnés, vues, watch time et CTR avec graphiques interactifs
- **🤖 Analyse IA** — Recommandations stratégiques personnalisées via Genkit (Gemini / Groq)
- **💡 Idées de contenu** — Génération d'idées de vidéos optimisées pour votre niche
- **🔍 Détection de niches** — Classification automatique de vos vidéos par thématique
- **📅 Calendrier éditorial** — Planning de publication suggéré par l'IA

## 🛠️ Stack Technique

| Couche | Technologie |
|--------|-------------|
| **Frontend** | Angular 19, Tailwind CSS, Spartan UI, ngx-charts |
| **Backend** | Node.js, Express 5, Genkit |
| **IA** | Google Gemini, Groq (Llama 3.3 70B) |
| **Base de données** | Supabase (PostgreSQL + Auth + RLS) |
| **Auth YouTube** | Google OAuth 2.0, YouTube Data API v3 |
